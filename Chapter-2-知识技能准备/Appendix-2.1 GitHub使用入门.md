# 2.1 GitHub使用入门

## 介绍

GitHub是在软件开发产业中常用的版本控制软件。我们的开发项目将会存储在GitHub，并对其进行管理和修改。在正式着手进行产品开发前，请确保你已经熟悉GitHub的使用方式。尤其是对于Pull Request以及Branch的使用。
